# Tesla Data Wrangling Project

This project demonstrates **data immersion and wrangling** using Python and Pandas.  
The dataset used is Tesla stock data from 2010 to 2025.

## Features
- Load and explore dataset (immersion)
- Clean and organize data (wrangling)
- Save cleaned dataset for further analysis

## Requirements
- Python 3.14+
- Pandas library
